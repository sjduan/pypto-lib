# Phase 0: 前置验证

> 在进入设计阶段前，必须完成算子类型识别和 Shape 验证。

---

## 0.1 算子类型识别

### 判断流程

```
公式分析 → 运算类型 → Shape 检查 → 算子类型
```

### 常见算子类型

| 类型 | 公式特征 | Shape 特征 | 推导方法 |
|------|---------|-----------|---------|
| **Broadcast** | 逐元素（+, -, *, /） | 输入 shape 不同 | 维度对齐 → 取 max |
| **矩阵乘法** | @, matmul | [M,K] @ [K,N] | 矩阵乘法规则 |
| **归约** | sum, max, min, mean | 输出维度减少 | 沿指定维度折叠 |
| **卷积** | conv2d, conv3d | 特殊 kernel 规则 | 卷积公式计算 |
| **拼接** | concat, stack | 沿某维度合并 | 指定维度相加 |
| **索引** | gather, scatter | 索引操作 | 按索引规则 |

## 0.2 Broadcast Shape 推导（最常见）

### 判断条件

三个条件同时满足：

| 条件 | 说明 |
|------|------|
| 1. 逐元素运算 | 公式包含 +, -, *, / 等逐元素操作 |
| 2. Shape 不同 | 输入 tensor 的 shape 不完全相同 |
| 3. 可兼容 | 每个维度满足：相等 或 其中一个为1 |

### 推导步骤

```
Step 1: 维度对齐
        从右向左补1，使所有 tensor 维度数相同

Step 2: 逐维比较
        每个维度取 max(各输入该维度值)

Step 3: 输出 Shape
        各维度最大值组成的 tuple
```

### 示例

**案例：Addcdiv 算子**

```
公式：output = x1 + value * (x2 / x3)

输入 Shape：
  x1: [1024]
  x2: [1, 1, 2, 1]
  x3: [1]

判断：
  ✅ 逐元素运算（+, *, /）
  ✅ Shape 不同
  ✅ 可兼容

推导：
  Step 1 - 维度对齐（补1到4维）：
    x1: [1024]      → (1, 1, 1, 1024)
    x2: [1,1,2,1]   → (1, 1, 2, 1)
    x3: [1]         → (1, 1, 1, 1)

  Step 2 - 逐维取 max：
    dim0: max(1, 1, 1) = 1
    dim1: max(1, 1, 1) = 1
    dim2: max(1, 2, 1) = 2
    dim3: max(1024, 1, 1) = 1024

  Step 3 - 输出：
    [1, 1, 2, 1024]
```

### 常见错误

| 错误 | 正确做法 |
|------|---------|
| 凭直觉猜测输出 shape | 按步骤逐维推导 |
| 忽略维度对齐 | 先对齐再比较 |
| 不验证结果 | 用 numpy 验证 |

## 0.3 其他算子类型推导

### 矩阵乘法

```
公式：C = A @ B

规则：
  A: [M, K]
  B: [K, N]
  C: [M, N]

条件：A 的最后一维 == B 的倒数第二维
```

### 归约运算

```
公式：y = reduce(x, dim=d)

规则：
  x: [d0, d1, d2, ..., dn]
  y: 移除 dim=d 后的 shape

示例：
  x: [2, 3, 4], reduce(dim=1)
  y: [2, 4]  （移除了 dim1=3）
```

### 拼接运算

```
公式：y = concat([x1, x2], dim=d)

规则：
  x1: [d0, d1, ..., dn]
  x2: [d0, d1, ..., dn]  （除 dim=d 外，其他维度相同）
  y: dim=d 的值相加

示例：
  x1: [2, 3, 4], x2: [2, 5, 4], concat(dim=1)
  y: [2, 8, 4]  （3+5=8）
```

## 0.4 验证方法

### 有 numpy 可用时（推荐）

```python
import numpy as np

# 构造输入
x1 = np.random.randn(1024).astype(np.float32)
x2 = np.random.randn(1, 1, 2, 1).astype(np.float32)
x3 = np.random.randn(1).astype(np.float32)

# 按公式计算
output = x1 + value * (x2 / x3)

# 验证 shape
print(f"输出 shape: {output.shape}")
```

### 无 numpy 时

1. 手动按步骤推导
2. 交叉验证（多种方法对比）
3. 查阅官方文档确认
