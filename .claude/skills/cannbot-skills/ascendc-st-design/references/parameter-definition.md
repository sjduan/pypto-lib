# 算子参数定义规范

## 参数类型定义

### aclTensor类型

| 字段名 | 类型 | 说明 | 示例 |
| ------ | ---- | ---- | ---- |
| name | string | 参数名称 | "self" |
| type | string | 固定值 | aclTensor |
| required | bool | 是否必须 | true |
| io_type | string | 输入输出类型：input/output | "input" |
| format | string | 数据格式 | "ND" |
| dimensions | list[int] | 维度信息 | [1, 2] |
| dtype_with_ranges | list[object] | 数据类型与值配置 | 见下文 |

**dtype_with_ranges 配置规范：**

| 字段名 | 类型 | 必填 | 说明 | 示例 |
| ------ | ---- | ---- | ---- | ---- |
| dtype | string | 是 | 数据类型（BFLOAT16/FLOAT32/FLOAT16等） | "float32" |
| value_range | list[list] | 否 | 输入tensor的取值范围（含特殊值），未指定时使用默认值 | [[-1,1],[0,0],["inf","inf"]] |

### aclScalar类型

| 属性 | 类型 | 说明 | 示例 |
| ---- | ---- | ---- | ---- |
| name | string | 参数名称 | "beta" |
| type | string | 固定值 | aclScalar |
| required | bool | 是否必须 | true |
| io_type | string | 输入输出类型：input/output | "input" |
| is_enum | bool | 是否枚举 | false |
| dtype_with_values | list[object] | 数据类型与值配置 | 见下文 |

**dtype_with_values配置规范：**

| 字段名 | 类型 | 必填 | 说明 | 示例 |
| ------ | ---- | ---- | ---- | ---- |
| dtype | string | 是 | 数据类型（BFLOAT16/FLOAT32/FLOAT16等） | "float32" |
| value_range | list[list] | 否 | 输入scalar的取值范围（含特殊值），is_enum为true时无该字段 | [[-1,1],[0,0],["inf","inf"]] |
| value | list[dtype_value] | 条件必填 | 枚举值列表，is_enum为false时无该字段 | [0, 1, 2, 3] |

### int8_t类型

| 属性 | 类型 | 说明 | 示例 |
| ---- | ---- | ---- | ---- |
| name | string | 参数名称 | "cubeMathType" |
| type | string | 固定值 | int8_t |
| required | bool | 是否必须 | true |
| io_type | string | 输入输出类型：input/output | "input" |
| is_enum | bool | 是否枚举 | false |
| dtype_with_values | list[object] | 数据类型与值配置 | 见下文 |

**dtype_with_values配置规范：**

| 字段名 | 类型 | 必填 | 说明 | 示例 |
| ------ | ---- | ---- | ---- | ---- |
| dtype | string | 是 | 数据类型 | "int8" |
| value_range | list[list] | 否 | 参数取值范围（含特殊值），is_enum为true时无该字段 | [[-1,1],[0,0],[5,5]] |
| value | list[dtype_value] | 条件必填 | 枚举值列表，is_enum为false时无该字段 | [0, 1, 2, 3] |

## io_type 字段说明

| 取值 | 说明 | 测试处理方式 |
| ---- | ---- | ------------ |
| input | 输入参数，参与算子计算 | 需生成测试数据，定义取值范围和特殊值 |
| output | 输出参数，存储计算结果 | 无需生成数据，无需定义取值范围和特殊值 |

### 特殊值表示

特殊值（如 inf, -inf, nan）使用范围表示：
- `["inf", "inf"]` 表示覆盖特殊值 无穷大
- `["-inf", "-inf"]` 表示覆盖特殊值 负无穷大
- `["nan", "nan"]` 表示覆盖特殊值 NaN
- `[0, 0]` 表示覆盖特殊值0

## 参数定义模板

```yaml
parameters:
# Tensor 类型参数（输入）
- name: <参数名>
  type: aclTensor
  required: <true|false>
  io_type: input
  format: "<格式>"
  dimensions: [<维度1>, <维度2>, ...]
  dtype_with_ranges:
    - dtype: <数据类型1>
      value_range: [[<min>, <max>], [<min>, <max>], ...] 
    - dtype: <数据类型2>
      value_range: [[<min>, <max>], ...]

# Tensor 类型参数（输出）
- name: <参数名>
  type: aclTensor
  required: <true|false>
  io_type: output
  format: "<格式>"
  dimensions: [<维度1>, <维度2>, ...]
  dtype_with_ranges:
    - dtype: <数据类型1>
    - dtype: <数据类型2>

# 非Tensor枚举类型参数
- name: <参数名>
  type: <int8_t|aclScalar|...>
  required: <true|false>
  io_type: input
  is_enum: true
  dtype_with_values:
    - dtype: <数据类型>
      value: [枚举值1, 枚举值2, ...]  # 必填

# 非Tensor非枚举类型参数
- name: <参数名>
  type: <aclScalar|int8_t|...>
  required: <true|false>
  io_type: input
  is_enum: false
  dtype_with_values:
    - dtype: <数据类型1>
      value_range: [[<min>, <max>], ...]
    - dtype: <数据类型2>
      value_range: [[<min>, <max>], ...]
```
