# 算子验收标准和用例级别规范

## 1. 算子验收标准

| 分类 | 验收标准 |
| ---- | -------- |
| 精度 | fp16:diff_thd/pct_thd为0.001；<br/>bf16:diff_thd/pct_thd为0.004；<br/>int/uint类型：diff_thd/pct_thd为0；<br/>其余类型：diff_thd/pct_thd为0.0001 |
| 性能 | 同算力带宽下，竞品A100/H800取最优：<br/>大模型网络算子平均1.2x，单case底线0.8x；<br/>全量网络算子平均1.0x，单case底线0.8x；<br/>随机泛化单case底线0.5倍；<br/>无竞品算子，单case性能不低于910B |

## 2. 用例级别标准

| 级别 | 定义 | 覆盖目标 | 用例数 |
| ---- | ---- | -------- | ------ |
| L0 | 门槛用例 | 核心功能直通 | <=200 |
| L1 | 功能/精度/性能用例 | 参数BC组合测试，正常+典型边界，DFX基准测试 | 500~700 |
| L2 | 异常用例 | 异常测试 | <=50 |
